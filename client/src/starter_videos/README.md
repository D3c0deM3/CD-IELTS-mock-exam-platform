# Starter Videos

This folder contains the instruction videos for each IELTS test section.

## Required Videos

1. **listening_starter.mp4** - Listening section instructions and test format overview
2. **reading_starter.mp4** - Reading section instructions (planned)
3. **writing_starter.mp4** - Writing section instructions (planned)

## Video Specifications

- **Format**: MP4 (H.264 video codec)
- **Resolution**: 1280x720 (HD) or higher
- **Duration**: 3-5 minutes recommended
- **File Size**: 50-200 MB

## Usage

Place your video files in this folder. The app will automatically load them in the respective starter screens.
